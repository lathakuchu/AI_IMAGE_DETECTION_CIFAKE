🖼️ AI Image Detection System (CIFAKE)

📌 Project Overview

The AI Image Detection System (CIFAKE) is a deep learning project developed to classify images as Real or AI-generated (Fake). The model is trained on the CIFAKE dataset using a Convolutional Neural Network (CNN) and deployed with Gradio on Hugging Face Spaces.

---

🎯 Objective

The goal of this project is to build an intelligent system that can accurately identify whether an uploaded image is genuine or AI-generated, helping improve digital content verification.

---

🚀 Features

- Upload an image for analysis.
- Classifies the image as Real or Fake.
- Displays the prediction confidence score.
- User-friendly Gradio web interface.
- Deployed on Hugging Face Spaces.

---

🛠️ Technologies Used

- Python
- TensorFlow
- Gradio
- NumPy
- Pillow (PIL)
- Matplotlib
- Hugging Face Spaces

---

📂 Dataset

Dataset: CIFAKE

The CIFAKE dataset contains both real and AI-generated images used to train and evaluate the deep learning model.

▶️ How to Run

1. Clone the repository.
2. Install the required packages:

pip install -r requirements.txt

3. Run the application:

python app.py

4. Open the Gradio interface in your browser.
